## 🛸 城乡无人机综合服务平台

<div class="project-meta">
<span>📅 2025.12 ~ 至今</span>
<span>👤 全栈开发 + 产品设计</span>
</div>

**项目概述**

面向系统管理员、站点管理员、飞手、用户四类角色的 B 端管理平台，覆盖无人机租借、航拍、表演、货运、喷洒、清洗等多元业务模块。

**技术栈**

<div style="display: flex; flex-wrap: wrap; gap: 4px;">
  <span class="skill-badge">Vue 3</span>
  <span class="skill-badge">Element Plus</span>
  <span class="skill-badge">ECharts</span>
  <span class="skill-badge">Spring Boot</span>
  <span class="skill-badge">MySQL</span>
  <span class="skill-badge">Redis</span>
  <span class="skill-badge">JWT</span>
  <span class="skill-badge">Git</span>
</div>

**核心成果**

- **飞手抢单功能**：设计并实现美团式抢单模式，优化飞手接单效率与平台派单逻辑
- **飞行执照审核流程**：完整的资质认证与审核链路，保障平台飞手准入质量
- **设备调度管理**：设备注册、维护、租赁、状态流转的全生命周期管理
- **数据可视化看板**：基于 ECharts 实现站点运营数据看板，涵盖订单量统计、设备利用率、收入趋势等指标
- **组件化架构**：封装 10 余个可复用业务组件（表格筛选、状态流转、文件上传等），提升开发效率
- **统一鉴权体系**：基于 JWT + axios 拦截器实现 token 自动刷新与角色级页面访问控制

## 🏥 移动端医院预约系统

<div class="project-meta">
<span>📅 2024.09 ~ 2025.01</span>
<span>👤 前端开发</span>
</div>

**项目概述**

基于 Vue 3 + Vant 构建的移动端 H5 医院预约系统，覆盖从科室选择到就诊完成的全流程。

**技术栈**

<div style="display: flex; flex-wrap: wrap; gap: 4px;">
  <span class="skill-badge">Vue 3</span>
  <span class="skill-badge">Vant</span>
  <span class="skill-badge">Vite</span>
  <span class="skill-badge">RESTful API</span>
</div>

**核心成果**

- **完整预约流程**：科室选择 → 医生排班 → 预约挂号 → 就诊记录查询，覆盖用户完整就医路径
- **组件化开发**：自定义封装日期选择器、就诊人管理等业务组件，减少重复代码，提升可维护性
- **交互细节打磨**：对接 RESTful API 完成数据交互，处理表单校验、加载状态、异常提示等细节

## 🎓 高考志愿管理系统

<div class="project-meta">
<span>📅 2023.07 ~ 2023.09</span>
<span>👤 前端开发</span>
</div>

**项目概述**

基于 React + Ant Design 搭建的高考志愿填报辅助系统，帮助考生进行院校查询与志愿推荐。

**技术栈**

<div style="display: flex; flex-wrap: wrap; gap: 4px;">
  <span class="skill-badge">React</span>
  <span class="skill-badge">Ant Design</span>
  <span class="skill-badge">JavaScript</span>
  <span class="skill-badge">MySQL</span>
</div>

**核心成果**

- **院校查询**：支持多维度筛选，帮助考生快速定位目标院校

- **数据库实践**：通过编写 MySQL 存储过程实现前后端数据交互，深入理解数据层设计
